# Hub de Dashboards TI — Field Service Operations

Suite de dashboards operacionais para field service de TI no varejo
brasileiro, gerados a partir de uma base real de chamados (17 meses,
5.006 registros) por uma engine de cálculo determinística.

## Ver ao vivo

Depois de publicar com GitHub Pages (veja abaixo), o link principal fica:

`https://SEU-USUARIO.github.io/NOME-DO-REPO/`

## Conteúdo

- `index.html` — página de apresentação
- `D1_VisaoOperacional_ClientesDiversos.html` — visão operacional (N1/N2/N3, parceiros, reembolsos, lab)
- `D2_VisaoGestores_ClientesDiversos.html` — visão gestores (produtividade agregada, saúde do contrato, comparativo anual)

## Como funciona

Cada dashboard é um arquivo HTML autocontido: a base de dados, a engine
de cálculo e a interface estão todos no mesmo arquivo. Tudo roda no
navegador — nada é enviado a um servidor.
